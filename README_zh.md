## 🤔 什么是 Grok Imagine？

**Grok Imagine** 是 **xAI** 推出的视频生成模型，支持文字生成视频、图片生成视频和视频编辑功能，深度集成在 Grok 聊天界面和 X（Twitter）平台中。

**核心特点：**
- 🎥 **文字转视频** — 通过文字描述生成高质量视频
- 🖼️ **图片转视频** — 将静态图片转化为动态视频
- 📹 **视频编辑** — 用自然语言编辑已有视频
- 🔊 **自动配音** — 自动配音和背景音乐生成
- 📐 **720p+ 分辨率**，时长 10-15 秒
- ⚡ **快速生成** — 20 秒内完成视频生成

**如何使用 Grok Imagine：**
- **Grok 聊天** — 在 [grok.com](https://grok.com) 或 xAI 应用中使用（提供免费版）
- **X/Twitter** — 直接在 X 平台中内置使用，面向所有用户
- **xAI API** — 通过 [xAI API](https://x.ai/api) 进行程序化调用
