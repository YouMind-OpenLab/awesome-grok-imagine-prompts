## 🤔 什么是 Grok Imagine？

**Grok Imagine** 是 **xAI 的 Aurora 模型**驱动的图像生成能力，集成在 Grok AI 助手中。它可以在 X（Twitter）平台上直接通过文本提示词生成令人惊艳的逼真和艺术图片。

**Key Features:**
- 🎨 **文本生成图片** — 根据文字描述生成图片
- 🖌️ **风格控制** — 支持从照片写实到插画等多种艺术风格
- 📸 **照片写实** — 生成与真实照片难以区分的高度逼真图片
- 🧑‍🎨 **角色设计** — 创建一致的角色和肖像
- 📐 **高分辨率** — 生成细节丰富的高质量图片
- 🐦 **X 平台集成** — 原生集成在 X（Twitter）的 Grok 中
