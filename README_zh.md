## 🤔 什么是 Grok Imagine？

**Grok Imagine** 是 **xAI** 推出的视频生成模型，集成在 X（Twitter）平台的 Grok AI 助手中。它可以通过文本提示词和图片生成高质量、电影级的视频。

**Key Features:**
- 🎬 **文本生成视频** — 根据文字描述生成视频
- 🖼️ **图片生成视频** — 将静态图片转化为动态视频
- ✂️ **视频编辑** — 编辑和转换现有视频
- 🔊 **自动音频** — 自动配音和背景音乐
- 📐 **720p+ 分辨率** — 每个片段 10-15 秒时长
- ⚡ **快速生成** — 20 秒内完成生成
