## 🤔 什么是 Grok Imagine？

**Grok Imagine** 是 **xAI** 推出的视频生成模型，集成在 X（Twitter）平台的 Grok AI 助手中。它可以通过文本提示词和图片生成高质量、电影级的视频。

**Key Features:**
- 🎬 **文本生成视频** — 根据文字描述生成视频
- 🖼️ **图片生成视频** — 将静态图片转化为动态视频
- ✂️ **视频编辑** — 编辑和转换现有视频
- 🔊 **自动音频** — 自动配音和背景音乐
- 📐 **720p+ 分辨率** — 每个片段 10-15 秒时长
- ⚡ **快速生成** — 20 秒内完成生成

### 详细的 10 秒 Grok 视频生成提示 (无缝衔接)

![日本語](https://img.shields.io/badge/lang-日本語-green)

> 一个高度详细的、多场景的视频提示，适用于 Grok Imagine，旨在无缝衔接之前 10 秒的视频。它详细说明了运镜、灯光效果（一道金色/蓝绿色光芒的爆发）、场景过渡（到一个奇幻仙境）、角色动作（女孩、鹿、仙子奔跑）以及技术参数（照片级真实感、8K-16K、60fps、无淡出）。

#### 📝 提示词

