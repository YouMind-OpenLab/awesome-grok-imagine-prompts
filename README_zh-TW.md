## 🤔 什麼是 Grok Imagine？

**Grok Imagine** 是 **xAI** 推出的視頻生成模型，集成在 X（Twitter）平台的 Grok AI 助手中。它可以通過文本提示詞和圖片生成高質量、電影級的視頻。

**Key Features:**
- 🎬 **文本生成視頻** — 根據文字描述生成視頻
- 🖼️ **圖片生成視頻** — 將靜態圖片轉化為動態視頻
- ✂️ **視頻編輯** — 編輯和轉換現有視頻
- 🔊 **自動音頻** — 自動配音和背景音樂
- 📐 **720p+ 分辨率** — 每個片段 10-15 秒時長
- ⚡ **快速生成** — 20 秒內完成生成
