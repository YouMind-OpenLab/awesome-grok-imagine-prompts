## 🤔 Grok Imagine とは？

**Grok Imagine** は **xAI** が開発した動画生成モデルで、X（Twitter）プラットフォーム上の Grok AI アシスタントに統合されています。テキストプロンプトや画像から高品質でシネマティックな動画を生成します。

**Key Features:**
- 🎬 **テキストから動画** — テキスト説明から動画を生成
- 🖼️ **画像から動画** — 静止画をダイナミックな動画に変換
- ✂️ **動画編集** — 既存の動画を編集・変換
- 🔊 **自動音声** — 自動吹き替えとBGM
- 📐 **720p+解像度** — クリップあたり10-15秒
- ⚡ **高速生成** — 20秒以内で生成完了

### タンクバイク変形動画プロンプト

![English](https://img.shields.io/badge/lang-English-blue)

> Grok Imagine 用の複雑な多段階ビデオプロンプトで、映画のようなトランジションを詳細に記述します。最初の画像の説明（プロンプトではありません）から始まり、未来的なオートバイに乗った男性が武装したタンクバイクのハイブリッドに乗った重装甲の人物に変身し、激しいアクションシーンで最高潮に達するビデオシーケンスを記述します。

#### 📝 プロンプト

```
戦車バイクに乗った男のイメージから始まり、カメラは高速で前方に追跡する。バイクの後部トラックが濡れたアスファルトをかき混ぜ、水しぶきと濃い白煙を巻き上げる。男はハンドルバーをしっかりと握り、風に身を任せる。背景の都市は動きでぼやけている。

ライダーの背中に機械的な装甲板が現れ始める。バイクのシャーシ前面に、ロボットの目のように光る赤い円形のレンズが点灯する。霧の立ち込める都市を加速するにつれて、バイクのエンジンは内部の赤い熱で輝く。

ライダーは背中に装甲をまとい、重厚な機械式外骨格の腕と肩がカチッと音を立てて装着される。スーツが固定されると、ライダーは腕を上げる。バイクの右側から大きなガトリング砲の銃身が現れる。火花を散らす金属とギアの動きが見える。

洗練された黒いロボットヘルメットが男の頭にカチッと閉まり、彼の顔を完全に隠す。バイクのフロントエンドが拡張し、2つ目のガトリング砲の銃身が所定の位置に回転する。ライダーは前かがみになり、機械と一体化する。

彼の全身は今や重厚で未来的な黒い装甲板に包まれている。バイクはさらに大きく、より攻撃的になり、複数の銃口が開く。カメラがわずかに横にパンし、戦車バイクのハイブリッドが道を轟音を立てて進むその巨大なスケールを示す。

バイクのガトリング砲が急速に回転し始め、オレンジ色の炎と曳光弾の連続的な流れを発射する。後部のポッドからロケットが発射され、煙の軌跡を残す。バイクが戦場を駆け抜けるにつれて、周囲は爆発と炎に包まれる。ハイオクタンなアクション、カメラの揺れ、激しいマズルフラッシュ、映画のような破壊。
```
